== Description ==

Simple to use Unit Converter Plugin for WordPress. It displays both in Metric and Imperial units. For the time, you can use the following categories:

    Length / Distance Converter
    Temperature Converter
    Time Converter
    Weight Converter
    Area Converter
    Speed Converter

The plugin creates a shortcode so you can display these converters either in posts / pages or inside a sidebar within a text widget.

New features in version 1.1.1

* Fixed working with jQuery 1.9+

== Installation ==

1. Upload `oppso-unit-converter` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. A page is created under settings where you can see a list of shortcodes that you can use inside posts / pages or in a sidebar as a text widget.
